32.1 Methods

32.1.1 Data Sources

River gauge data from USGS gauge 01358000 was obtained from USGS water data.

32.1.2 Data Analysis

Mean annual flow rate was calculated by averaging all flow rate data for each year. Cubic feet per second were converted to cubic meters per second. Attached code shows this process in detail.

A linear trend and a nonlinear GAM were calculated for the resulting annual mean flow rate time series. Attached code shows this process in detail.
